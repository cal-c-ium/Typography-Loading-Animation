# Typography-Loading-Animation
A web based loading animation made from GSAP, and based off code from the osmo.supply demo.


# How to use:
1. copy and paste the HTML code into your editor.
2. drop and paste the CSS file into your folder. Rename the files and its corresonding link elements.
3. copy and paste the JS files into your folder. Rename the files and its corresonding link elements.

The GSAP Files are from the gsap-public.zip from the official GSAP Website (https://gsap.com/).
